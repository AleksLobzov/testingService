# testingService

Сервис автоматизированного тестирования, позволяющий выполнять следующие функции:

<ol>
<li>
Средствами админки Django:
<ol>
<li>
Регистрация пользователя в системе.
</li>
<li>
Создание тестов (тесты, вопросы, ответы - Subject, Question, Answer).
</li>
</ol>
</li>
<li>
Средствами интерфейса, реализованного на базе template-машины Django:
<ol>
<li>
Авторизация пользователя (страница login.html).
</li>
<li>
Выбор теста авторизованным пользователем (страница index.html).
</li>
<li>
Ответ на вопрос (страница subject.html).
</li>
<li>
Подстчет результатов тестирования (страница subject.html)
</li>
</ol>
</li>
</ol>
